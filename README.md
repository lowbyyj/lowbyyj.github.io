# lowbyyj.github.io

Public GitHub Pages repository for <https://lowbyyj.github.io>.

This site is Yongjae Yoo's personal CV, research profile, career archive, and
future project/artifact showroom. The repository is intentionally public so the
site can support professional visibility, recruiting, headhunter discovery,
research identity, and future startup opportunities.

## Public Repository Rules

- Keep this repository public unless the owner explicitly says otherwise.
- Do not commit secrets, tokens, API keys, credentials, `.env` files, private
  notes, drafts, local config, logs, caches, build outputs, or company-internal
  details.
- Career, employer, team, role, research, and publication information may be
  intentionally public when the owner considers it legally and professionally
  appropriate.
- Do not over-sanitize CV or career-forward content just because the repository
  is public.
- Do not disclose confidential company information, unreleased technical
  details, private documents, or unreviewed artifacts.

## Site Structure

This is a Jekyll-based GitHub Pages site.

- `_config.yml` controls site metadata and Jekyll settings.
- `index.md` contains the current homepage/CV content.
- `contact.md` contains the contact page.
- `_includes/` contains active shared head/header/footer partials.
- `_layouts/` contains page/post/archive layout templates.
- `assets/css/main.scss` controls the main styling.
- `assets/images/` and `assets/my-favicon/` contain public site assets.

As of this note, there is no `Gemfile` in the repository. On this Windows
machine, local `ruby`, `bundle`, and `jekyll` commands may not be installed, so
local builds may require a Ruby/Jekyll setup before they can be run.
